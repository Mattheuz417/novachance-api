# 🚀 NovaChance API

Sistema desenvolvido para gerenciamento de oportunidades voltadas para jovens em situação de vulnerabilidade social.

Projeto realizado como atividade extensionista da disciplina **Tema Integrador – Projeto de Sistemas III** da UNIG.

---

# 📌 Objetivo

O projeto tem como objetivo criar uma plataforma para cadastro e gerenciamento de:
- Usuários
- Cursos
- Empregos
- Mentorias
- Oportunidades profissionais

Buscando promover inclusão digital e acesso à informação através da tecnologia.

---

# 🛠️ Tecnologias Utilizadas

- Node.js
- Express.js
- MySQL
- XAMPP
- bcrypt
- dotenv
- HTML
- CSS
- JavaScript
- Git
- GitHub

---

# 🧱 Arquitetura

O sistema foi desenvolvido utilizando arquitetura MVC:

- Models → comunicação com banco de dados
- Controllers → lógica da aplicação
- Routes → gerenciamento de rotas
- Frontend → interface visual integrada à API
